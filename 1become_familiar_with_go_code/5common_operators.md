# common operators

**主要了解清位运算符(`&^`)和位反(`^`)**

二元运算符:
+ 2个基本数值类型: `+`, `-`, `*`, `/`
+ 2个基本整数数值类型: `%`

**位运算符**:
+ `&`, `|`, `^`(异或), **`&^`(清位, 1111 &^ 1100 = 0011)**
+ `<<`, `>>`, 左操作数为整数, 右操作数为非负整数

一元运算符:
+ `+`(取正数), `-`(取负数), **`^`(位反, 按位取反)**

更多关于常量表达式

*常量表达式的顺序有可能影响到最终的估值结果*

```golang
var x = 1.2 + 3/2 // 2.2

const x = 3/2*0.1 // 0.1
const y = 0.1*3/2 // 0.15
```
