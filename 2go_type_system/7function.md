# function

```golang
// 函数字面量: func(a int, b, c string) (x, y int, z bool)
// 函数原型: `func aFunction(n int) (result int)`
// 函数声明 = 函数原型 + 函数体
```

***所有的函数类型都属于不可比较类型()***

***所有函数调用的传参均属于值复制***

***函数调用作为参数***
```golang
// 1. 基本规则: 返回值个数不为0, 可以用作另1函数实参, 但不可和其它实参混合
// 2. 语法糖: 刚好返回1个结果, 当作单独实参, **可以与其它实参混合使用在同1函数实参列表中**
// 实际表现就是: 返回1个结果的函数调用可以和其它实参在参数列表中混合使用,
//   否则(返回值个数大于1)不能与其它实参混合在1个参数列表中使用
```
